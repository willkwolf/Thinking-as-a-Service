# Thinking as a Service

Sitio interactivo de consultoría estratégica para alta dirección: recorra el iceberg desde el **ruido** (IA, VUCA, BANI, TUNA) hasta el **Playbook** de reducción de complejidad, con evidencia visual y mapas Kumu.

**Sitio en vivo:** [willkwolf.github.io/Thinking-as-a-Service](https://willkwolf.github.io/Thinking-as-a-Service/)

## Diferencial

Antes de automatizar con IA, se describe el sistema: qué complejidad añade valor y qué es ruido que destruye margen (~10,2% de valor para accionistas). Simplicidad operativa con **Complexity Diagnosis** y **Complexity Reduction Strategy**, liderado por **PhD Andrés López Astudillo**.
